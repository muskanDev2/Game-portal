# Game Portal

**Game Portal** is a beginner-level web project built using **HTML, CSS, JavaScript, and Bootstrap**. This project combines the foundational web development knowledge I had so far and allows me to practice **clean coding principles** while incorporating new concepts learned recently, especially **HCI (Human-Computer Interaction) principles**.

## Purpose

The project was created to **practice and consolidate beginner web development skills** and to polish them by applying the HCI principles I learned in the last semester. It demonstrates:

- Structuring web pages with **HTML**
- Styling and layout with **CSS** and **Bootstrap**
- Adding interactivity using **JavaScript**
- Applying **clean coding practices** and **HCI concepts** to improve usability

## Features

- Simple games like **Tic Tac Toe** and **Coin Flip**
- Beginner-friendly and intuitive interface
- Responsive design using **Bootstrap**
- Incorporates **HCI principles** to enhance user experience

## Learning Outcome

Through this project, I was able to:

- Combine **HTML, CSS, and JavaScript** in a functional project
- Apply **HCI principles** to improve design and usability
- Practice **clean coding and project organization**
- Strengthen my beginner-level knowledge and polish it with new concepts
